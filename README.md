# Clams QR

A simple script for creating Clams QR png files for a list of URL's

## Create QR's

1. Install the deps:

```bash
yarn
```

2. Add a new line separated list of urls to a `./url` file. See `./example.urls` for an example file.

3. Run the script

```bash
yarn qr
```

4. Check out your sweet QR png files in `./qrs`
